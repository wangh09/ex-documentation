## Matching Engine

go into cryptoexchange-build/build and run bash build-jar.sh, after a successful build you will obtain 

```
crypto-api.jar
crypto-config.jar
crypto-sequence.jar
crypto-quotation.jar
crypto-match.jar
hd-hot-wallet.jar
```

at each sub-projects.
